== Description ==

Gutena Video Lightbox is a WordPress Block that allows you to add a video in a popup window that goes over the website content. To create a lightbox, Just upload a display image, then put a video URL of Youtube / Vimeo.

[youtube https://www.youtube.com/watch?v=pY-x4erbu4I]

== Features ==

* This plugin can play the videos from Youtube and Vimeo
* Complete control over the look and feel
* User-friendly interface, so you don’t need any coding skills to make it beautiful
* Lightweight and fast, so it won’t slow down your website
* Pre-made layouts to choose from

== Installation ==

1. Visit 'Plugins > Add New'.
1. Search for 'Gutena Video Lightbox' and install it.
1. Or you can upload the `gutena-lightbox` folder to the `/wp-content/plugins/` directory manually.
1. Activate Gutena Video Lightbox from your Plugins page.
